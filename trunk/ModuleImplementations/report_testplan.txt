Test Plan
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

To test the Report shell script, we had to put it under rigorous manual testing, as
automated regression testing was not a viable option (would need to edit the original
component under test). Instead, we had just executed the program and compared the summary.txt
generated by the program with what we expected it to yield. The directory stucture was modified
as follows, with their assignment statuses:
